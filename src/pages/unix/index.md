# Unix Commands
> Don't be afraid of the terminal!

## Files

### Creating a File

```
touch file.py
```

### Moving a File

```
mv file.py another/folder
```

### Deleting a File

```
rm file.py
```

### Contents of a File

```
cat file.py
```

### Editing a File

```
nano file.py
```

or if you are particularly brave:

```
vi file.py
```

## Folders

### Creating a Folder

```
mkdir folder
```

### Contents of a Folder

```
ls folder
```

### Deleting a Folder

```
rm -rf folder
```
